## 📖 Overview

Gesture Classifier AI is a real-time mobile gesture recognition application that uses smartphone accelerometer sensor data to identify user movements and classify them into predefined gestures.

The application continuously captures motion sensor readings, processes the incoming data, and predicts gestures such as Circle, Wave, and Idle while displaying confidence scores and live sensor visualizations.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Gesture-Classifier-AI.git
```

Navigate to project directory:

```bash
cd Gesture-Classifier-AI
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
